# Portfolio Website - Nathan Ahnaf Hafiz

Selamat datang di repositori portofolio pribadi saya! Website ini dirancang untuk menampilkan profil, keahlian, pengalaman, serta proyek-proyek yang telah saya kerjakan sebagai seorang **Web Developer**.

Website ini dibuat agar modern, interaktif, cepat, dan sepenuhnya responsif di berbagai perangkat.

## 🚀 Fitur Utama

- **Desain Modern & Responsif:** Tampilan clean dengan dark mode dan tata letak yang menyesuaikan ukuran layar (Mobile, Tablet, Desktop).
- **Interaktif:** Dilengkapi dengan animasi halus untuk meningkatkan pengalaman pengguna (UX).
- **Halaman Dinamis:** Berorientasi pada komponen satu halaman (*Single Page Application* style menggunakan navigasi *anchor*).
- **Form Kontak:** Formulir yang siap menerima pesan atau tawaran kolaborasi.

## 🛠️ Teknologi yang Digunakan

### Frontend
- **HTML5** & **CSS3** (Layouting, Flexbox, Grid, Animasi)
- **JavaScript (ES6+)** (Interaktivitas dan Logika)

### Tools & Deployment
- **Git** & **GitHub** (Version Control)
- **VS Code** (Code Editor)
- **Figma** (UI/UX Design)

## 📁 Struktur Proyek yang Ditampilkan

Di dalam website ini, Anda dapat menjelajahi beberapa bagian utama:
1. **Beranda:** Salam pembuka dan ringkasan singkat profesi saya.
2. **Tentang Saya:** Cerita singkat mengenai fokus saya dalam membangun produk web.
3. **Keahlian:** Tech stack yang saya kuasai, mencakup Frontend, Backend dasar, Desain, dan Tools.
4. **Proyek:** Etalase proyek pilihan, seperti *Landing Page* minimalis dan *CHILL Movie Streaming App*.
5. **Kontak:** Formulir interaktif untuk terhubung langsung dengan saya.
